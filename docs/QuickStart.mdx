# شروع سریع (Quick Start)

در این بخش با مراحل اولیه طراحی و راه‌اندازی یک برنامک ساده برای پیام‌رسان ایتا آشنا می‌شوید. هدف این است که شما بتوانید در کمترین زمان، یک اپلیکیشن سبک و تعاملی بسازید و آن را در محیط ایتا تست کنید.

---

## پیش‌نیازها

برای شروع توسعه یک برنامه ، به موارد زیر نیاز دارید:

- یک دامنه معتبر با پشتیبانی از **HTTPS**
- یک محیط میزبانی برای فایل‌های وب اپلیکیشن (مانند Cpanel, Direct admin, Vercel، Netlify، GitHub Pages، یا سرور اختصاصی)
- دسترسی به پنل توسعه‌دهندگان برنامک در ایتا
- آشنایی مقدماتی با HTML، CSS و JavaScript

---

## مراحل ساخت برنامه

در کل مسیر ساخت برنامک شامل ۴ مرحله اصلی است:
1. [ثبت برنامه در سامانه ایتایار](#step1)
2. [ساخت وب اپلیکیشن](#step2)
3. [اجرای برنامه در محیط ایتا](#step3)

<Timeline>

  <TimelineItem title="مرحله 1 : ثبت برنامه در سامانه ایتایار" id="step1">
    برای اجرای برنامه در پیام‌رسان ایتا، ابتدا باید آن را در [سامانه ایتایار (EitaaYar)](https://eitaayar.ir) ثبت کنید.

    1. وارد سایت [ایتایار](https://eitaayar.ir) شوید.
    2. از منوی بالا یا نوار کناری، وارد بخش **برنامه‌ها** شوید.
    3. اگر اولین بار است که وارد این بخش می‌شوید، باید فرآیند **احراز هویت توسعه‌دهنده** را انجام دهید.
    4. پس از تایید، روی دکمه **افزودن برنامه** کلیک کنید.
    #### فرم ثبت برنامه
    در فرم بازشده، مشخصات برنامه را وارد می‌کنید:
    - **تصویر پروفایل:** PNG یا JPG برای پروفایل برنامه
    - **عنوان:** مانند «ماشین‌حساب ساده»
    - **نام کاربری برنامه:** مثل `my_app`
    - **وضعیت:** فعال یا غیرفعال بودن برنامه
    - **توضیحات برنامه:** شرح عملکرد برنامه
    - **عبارت دکمه:** مثل «شروع» یا «اجرا»
    - **آدرس مقصد:** آدرس HTTPS برنامه (مثل `https://yourdomain.com/app/index.html`)
    - **عکس شروع:** تصویر کارت برنامه
    - **پیام شروع:** پیام معرفی زیر کارت

    پس از تکمیل فرم و بارگذاری فایل‌ها، روی دکمه **ساخت برنامه** یا **ویرایش برنامه** کلیک کنید.

    اگر برنامه فعال باشد و آدرس مقصد معتبر باشد، برنامه برای تست و استفاده در ایتا در دسترس خواهد بود.

    :::info
    **توجه:** آدرس برنامک باید [HTTPS](https://parspack.com/blog/security/ssl/install-free-ssl-iis) باشد و **نباید ریدایرکت داشته باشد**.
    :::

    توضیح کامل این مرحله را می‌توانید در صفحه [ثبت و مدیریت برنامه](/docs/ManagementPanel) بخوانید.
  </TimelineItem>

            ساخت وب اپلیکیشن در آدرس مقصد ، جهت دسترسی به کیت توسعه ( SDK )

             **[کیت توسعه ( SDK )](Develop/JsSDK.md)** رابطی است بین برنامه یا برنامک و محیط پیام‌رسان ایتا که اجازه می‌دهد با امکانات ایتا (مثل بستن، تمام‌صفحه شدن و غیره) تعامل داشته باشد.

                نمونه وب اپلیکیشن :

      <CodeBlock language="html">{`
  <!DOCTYPE html>
  <html lang="fa">
  <head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>برنامک نمونه</title>
  <script src="https://developer.eitaa.com/eitaa-web-app.js"></script> //  جهت دسترسی به  کیت توسعه
  <style>
    body { font-family: sans-serif; text-align: center; padding-top: 50px; }
    button { font-size: 18px; padding: 10px 20px; }
  </style>
  </head>
  <body>
  <h1>سلام از برنامک من!</h1>
  <button onclick="window.Eitaa?.WebApp.close()">بستن برنامه</button>
  <script>
      // بررسی دسترسی به کیت توسعه
    if (window.Eitaa) {
      console.log("SDK در دسترس است");
    } else {
      console.log("SDK فعال نیست");
    }
  </script>
  </body>
  </html>
      `}</CodeBlock>

:::info
ℹ️ توجه: برنامه ها و برنامک‌ها در حال حاضر فقط از طریق نسخه **اندروید** و **نسخه دسکتاپ (ویندوز)** ایتا قابل اجرا هستند. نسخه iOS یا مرورگرهای دیگر پشتیبانی نمی‌شوند.
:::
    پس از ثبت موفق، برنامه را می‌توانید در محیط پیام‌رسان ایتا تست و اجرا کنید:
    1. وارد محیط برنامه شوید. لینک آن از شناسه کاربری برنامه ساخته می‌شود.
    2. کاربر با کلیک روی دکمه برنامه ، آن را در یک **WebView داخلی** مشاهده خواهد کرد.
    3. WebView به URL ثبت‌شده متصل شده و محتوای برنامه را اجرا می‌کند.
